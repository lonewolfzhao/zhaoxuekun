# chromego_merge

## 注意事项

套上warp可绕过chromego封锁的网站（P，X）

开启浏览器自带doh以及客户端tun模式也可绕过封锁，参考：[开启chrome自带doh](https://blog.misaka.rest/2023/10/30/enable-doh-ech)

## 如何修改为自己的warp节点
